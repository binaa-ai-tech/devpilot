# Changelog Entry

Add one line to `CHANGELOG.md` under the unreleased version:

```
### <Added | Changed | Fixed | Removed | Security>
- <TICKET-KEY>: <one-line user-facing description>
```

Examples:

```
### Added
- MSK-101: Tenant signup flow with email verification

### Fixed
- NSP-247: Cheque status now updates correctly on partial settlement

### Changed
- RAF-30: Draw service uses takeUntilDestroyed for cleanup (internal)
```
